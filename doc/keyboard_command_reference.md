# Emerald keyboard command reference
------------

## Movement 

| Action  | Keyboard mapping |
| ------- | ---------------: |
| Forward | W |
| Backward | S |
| Left | A |
| Right | D |
| Attach/detach spline movement | F |
| Toggle next spline | : |
| Toggle previous spline | ; |


## Rendering modes

| Mode    | Keyboard mapping |
| ------- | ---------------: |
| Forward | 1 |
| Deferred | 2 |
| Position buffer | 3 |
| Normal buffer | 4 |
| Albedo buffer | 5 |
| Specular intensity buffer | 6 |
| SSAO | 7 |

## GUI

| Action  | Keyboard mapping |
| ------- | ---------------: |
| Toggle anttweakbar on/of | T |
